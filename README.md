# Checklist - Práctica Formativa Obligatoria 1

## Descripción del Proyecto

Este es mi tercer portafolio personal, desarrollado como Trabajo Práctico. El sitio web muestra mi perfil profesional como desarrolladora Backend, destacando no solo mis herramientas técnicas, sino también mi enfoque en la resolución de problemas lógicos y el "pensamiento lateral". La estética es minimalista, inspirada en códigos y enigmas, utilizando un fondo crema con líneas gris oscuro y acentos amarillos, reflejando la idea de descifrar complejos rompecabezas digitales.

Layout y Organización: <br>
[x] Se ha organizado el layout (especialmente en la sección "tarjetas") utilizando Flexbox o Grid. <br>
[x] Redacta: ¿Qué ventajas encontraste al utilizar Flexbox o Grid en tu proyecto? <br>
**Respuesta:** Encontré enormes ventajas al usar CSS Grid para las grillas de tarjetas y películas. Con una sola regla (`grid-template-columns: repeat(auto-fit, minmax(...))`) logré un diseño 100% fluido que se adapta a cualquier tamaño de pantalla de forma automática, manteniendo las proporciones sin tener que escribir reglas complejas. Flexbox, por su parte, me resultó ideal para centrar elementos y alinear la navegación.

[x] Se han personalizado los estilos de tablas, botones, enlaces y formularios. <br>
[x] Se han ajustado las dimensiones de imágenes y contenedores utilizando unidades relativas (%, rem, vh). <br>
[x] Se ha implementado al menos una animación o transición (por ejemplo, efecto hover en tarjetas o botones). <br>
[x] Redacta: ¿Qué animación o transición implementaste y por qué consideraste que era adecuada para tu proyecto? <br>
**Respuesta:** Implementé un efecto `hover` en las tarjetas y películas usando `transform: translateY(-10px)` combinado con una sombra (`box-shadow`) asimétrica de color amarillo. Me pareció adecuada porque es sutil, no sobrecarga la vista y da la sensación física de que la tarjeta se "desbloquea" o se levanta como una pieza de rompecabezas, sumando dinamismo a la interfaz minimalista.
